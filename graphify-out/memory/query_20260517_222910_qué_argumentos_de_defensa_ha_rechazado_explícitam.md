# Q: ¿Qué argumentos de defensa ha rechazado explícitamente el TDLC o la CS en casos de colusión? Para cada argumento rechazado: sentencia que lo descarta, razón del descarte, y si existe algún caso posterior donde un argumento similar fue acogido.

## Answer

20 argumentos catalogados con citas textuales exactas de 7 sentencias (74/2008, 128/2013, 139/2014, 145/2015, 160/2017, 165/2018 CS, 171/2019). Solo ARG-128-C (ausencia de participación + mera expresión gremial) fue acogido (absolución). ARG-74-D (buena fe subjetiva con consulta previa) opera solo como atenuante de multa. El sorteo aleatorio no configura reparto de mercado. Archivo: graphify-out/notes/colusion-argumentos.md
