# Notas personales del investigador

Esta carpeta contiene anotaciones, hipótesis y observaciones propias.
**No son fuentes de autoridad jurídica.** Se incluyen en el grafo como nodos de análisis con `type: nota-personal`.

Para crear una nota nueva, usa la plantilla:

```yaml
---
type: nota-personal
author: sebchac
source: análisis propio
autoridad: ninguna
---
```
